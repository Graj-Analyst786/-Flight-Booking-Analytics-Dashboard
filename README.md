# -Flight-Booking-Analytics-Dashboard
This project analyzes flight booking data using Power BI to uncover trends in customer bookings, ticket pricing, payment methods, airline performance, and travel behavior. It also includes a NotebookLM Insight Card that summarizes key business findings for executives and decision-makers.
